# Documentation

The active documentation set is intentionally small:

- `llm-wiki.md`: Karpathy's original abstract pattern.
- `guide_llm_wiki_nashu.md`: an external implementation reference.
- `wiki-agent-architecture.md`: architecture authority for this repository.
- `roadmap.md`: module-level delivery plan.
- `evaluation.md`: evidence required before adding complexity.

Historical Knowledge Compiler documents were removed because they prescribed a
large semantic pipeline that no longer represents the project direction.
